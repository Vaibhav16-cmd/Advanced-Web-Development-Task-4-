# TASK-4: Advanced Web Development Projects

## Overview

This repository contains three web development projects built using HTML, CSS, and JavaScript as part of my Web Development Internship Program.

## Projects Included

### 1. Personal Portfolio Website

A responsive portfolio website showcasing personal information, skills, projects, and contact details.

**Features:**

* Responsive Navigation Bar
* Home Section
* About Section
* Skills Section
* Projects Section
* Contact Form
* Mobile-Friendly Design

### 2. To-Do List App with Local Storage

A task management application that allows users to add, complete, and delete tasks while saving data using Local Storage.

**Features:**

* Add Tasks
* Mark Tasks as Completed
* Delete Tasks
* Local Storage Support
* Responsive Design

### 3. Product Listing Page with Filtering and Sorting

An interactive product page where users can filter products by category and sort them by price or rating.

**Features:**

* Product Cards
* Category Filtering
* Price Sorting
* Rating Sorting
* Responsive Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage API
